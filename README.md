# Video Game Catalogue

A simple two-page full-stack application for browsing and editing video game records.

## Tech Stack

**Backend**
- ASP.NET Core (latest)
- EF Core
- SQL Server Express

**Frontend**
- Bootstrap
- Angular (latest)
- Angular Router

## Running the Application

### Backend
1. Open the backend project in Visual Studio
2. Run the project

### Frontend
1. Navigate to the frontend directory with PowerShell
2. Install dependencies: ```npm install```
3. Start the Angular app: ```npm start```
